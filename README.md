# Sesame

Sesame is a lightweight Craft CMS plugin that enables password-less authentication via magic links. Users receive a one-time, time-limited login link directly to their inbox, with no password required. This not only streamlines the login experience but also eliminates the risk of weak or compromised credentials. *Open Sesame* — just click the link and you're in.

## Requirements

This plugin requires Craft CMS 5.0.0 or later, and PHP 8.2 or later.
